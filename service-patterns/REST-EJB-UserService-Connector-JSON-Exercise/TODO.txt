Exercise: Service Connector
-------------------------------------------------------------------------------

How to Access the REST Service?
-------------------------------------------------------------------------------

Use the known REST-EJB-UserService implementation:

http://localhost:8080/REST-EJB-UserService/v1/users

	
Service Connector
-------------------------------------------------------------------------------
Implement a class called "UserServiceConnectorJSON" which implements the given 
interface "UserServiceConnector".
The class is responsible for handling the JSON based communication with the given
REST service.

You can use both Burp Suite and the "UserServiceConnectorTest" to verify your 
implementations.
